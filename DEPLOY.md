# Publicação no Render

1. Criar no GitHub o repositório público vazio `quatro-em-linha-torneio`.
2. Colocar neste repositório todos os ficheiros do projeto.
3. No Render, escolher **New > Blueprint** e selecionar o repositório.
4. Confirmar o serviço definido em `render.yaml`.
5. Definir `ADMIN_PIN` com o PIN privado do professor.
6. Depois de o serviço ficar **Live**, abrir `/admin.html`.

O endereço dos jogadores é público. O PIN protege apenas as operações do professor.
No plano gratuito, os dados podem desaparecer quando o serviço reinicia ou é novamente publicado.
